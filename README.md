# vendor-security-risk-assessment-lab
Third-party vendor security risk assessment project including questionnaire, risk scoring and vendor risk report.
## Scenario

This project simulates a third-party vendor security risk assessment conducted before onboarding external service providers.

Organizations rely on third-party vendors such as cloud providers, SaaS platforms, and payment processors. These vendors can introduce security risks if their controls are weak.

The assessment includes:

• Vendor security questionnaire
• Vendor risk scoring methodology
• Vendor risk assessment report template

The goal is to evaluate vendor security posture and identify potential risks before establishing a business relationship.


This project demonstrates a simplified third-party vendor security risk assessment process.

The project includes:

Vendor Security Questionnaire  
A questionnaire used to evaluate vendor security practices.

Vendor Risk Scoring  
A simple risk scoring sheet based on likelihood and impact.

Vendor Risk Assessment Report  
A template used to summarize vendor risk findings.

Purpose:
To demonstrate how organizations evaluate third-party security risks before engaging with vendors.
